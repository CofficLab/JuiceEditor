# JuiceEditor

一个强大富文本编辑器。 

目前仍处于开发中，尚不稳定。

## 快速开始

```html
<!DOCTYPE html>
<html lang="en">

<head>
  <script type="module" crossorigin src="./juice-editor/assets/index-23e26c6d.js"></script>
</head>

<body>
  <juice-editor>
    <h1>Say Hello to JuiceEditor</h1>
    <pre><code>console.log("ABC")</code></pre>
  </juice-editor>
</body>

</html>
```

## 文档
